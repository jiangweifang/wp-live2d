除了 WordPress 外掛之外，本專案還提供一個 Chrome / Edge 瀏覽器擴充功能 Live2D Web Canvas，讓您在任何網頁右下角都能掛一個看板娘。它與 WordPress 外掛是兩個獨立產品，登入帳號通用，但設定完全分開。

1. 它能做什麼？

  • 在您造訪的任何網站右下角掛一個 Live2D 看板娘（不讀取所在網頁的任何內容，僅依滑鼠位置做朝向動畫 —— 詳見隱私權政策）。
  • 開箱自帶幾款經典 V1 模型（22 娘 / 33 娘、Pio / Tia、Shizuku、艦これ Murakumo、超次元海王星等），下載到 IndexedDB 離線使用。
  • 登入付費帳號後，可以填自己的 modelAPI 載入 V2（Cubism 4+）模型。
  • 可調頭部錨點、縮放倍率、看板娘位置、畫布尺寸、提示氣泡頻率、工具列按鈕等。

2. 安裝

從 Chrome Web Store / Edge 附加元件商店搜尋 Live2D Web Canvas 安裝即可。安裝後位址列右側會出現擴充功能圖示，點開可開啟 popup（快速開關）或進入「選項」頁（完整設定）。

3. 登入（僅在使用 V2 時需要）

選項頁 → 登入 標籤 → 「使用網頁登入」。它會跳轉到 https://www.live2dweb.com/SingleLogin 完成登入，登入成功後頁面會自動關閉並把存取令牌寫入 chrome.storage.local。

  • 登入狀態僅本機，不會跨裝置同步。
  • 僅在您主動登入時才會發生網路請求，擴充功能不收集您造訪的網頁 URL / 標題。
  • 登出會清除本機令牌。

若您只想用免費的 V1 模型，無需登入。

4. 模型設定

選項頁 → 基本設定 → 模型 → 模型來源 是一個二選一下拉：

模型來源意義付費？
本機模型（V1）從擴充功能自帶的 V1 模型庫中選一個已下載的模型 + 一套貼圖。先去「V1 模型庫」標籤把想用的模型下載下來。免費
自訂 URL（付費）填一個您自己的 modelAPI，載入 V2（Cubism 4+）模型。兩種寫法與 WordPress 外掛一致：直連 .json 或目錄根 / + modelDir 列表。需要登入 + 付費

4.1 V1 模型庫

選項頁 → V1 模型庫 標籤：列出所有可用的 V1 模型（帶 64×64 縮圖、狀態徽章），點「下載」會把模型檔案存到瀏覽器的 IndexedDB（不會上傳到任何伺服器）。

  • 多貼圖型（如 22 娘 / 33 娘 / Pio / Tia）：單一 model.moc + 多套貼圖，下方 Texture ID 下拉是從 *.textures.json 解析出來的貼圖組。
  • 多面板型（如超次元海王星 / 艦これ / Shizuku Talk）：每個「面板」是獨立的子模型（各自的 model.moc），Texture ID 下拉裡列出的是子目錄名。

4.2 自訂 URL（V2）

登入付費帳號後，模型來源選「自訂 URL（付費）」，下方會出現 modelAPI 與 modelDir 兩個輸入。寫法與 WordPress 外掛完全一致 —— 看 V2 自訂模型教學的第 3 節。

目前瀏覽器擴充功能端沒有「快取至本機 / 防盜連」開關。擴充功能是直接從 modelAPI 拉模型載入，所以請保證 modelAPI 本身就支援 CORS（必須允許 chrome-extension://... 或 * 作為 Origin）。

5. 其他設定 tab

  • 基本設定：啟用開關、模型、頭部錨點 (headPos)、模型縮放、看板娘位置 (X/Y)、畫布尺寸。
  • 工具列設定：右下角小工具按鈕（截圖、切換、設定等）的開關。
  • 提示訊息選項：預設提示文案、觸發頻率。
  • 進階設定：開發除錯用，一般不用動。
  • V1 模型庫：上面已經講過。

6. 與 WordPress 外掛的關係

WordPress 外掛瀏覽器擴充功能
掛在哪裡您自己的 WP 站台使用者瀏覽器，所有網頁
誰來設定站長（一次設定全站生效）每位使用者在自己的瀏覽器設定
V1 模型來源外掛內建 + 遠端創意工坊擴充功能內建（IndexedDB 快取）
V2 自訂 URL付費 + 「快取至本機」可選付費，無本機快取（直連）
帳號同一個 live2dweb.com 帳號，但授權令牌互不共享，需要分別登入

7. 隱私

擴充功能不讀取您造訪的網頁 URL、內容、Cookie、表單或瀏覽歷史，不收集地理位置 / IP / 裝置指紋 / 健康資訊 / 通訊內容。完整的資料使用清單見 瀏覽器擴充功能隱私權政策。
