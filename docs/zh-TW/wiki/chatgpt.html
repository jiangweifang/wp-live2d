感謝您使用我的外掛。

功能與預設配額

  • 本外掛中提供了 ChatGPT 功能，您可以透過點擊機器人和 ChatGPT 互動。由於 ChatGPT 是付費 API，作者免費提供的預設通道有配額限制，不希望被濫用，所以預設程式碼中把 MaxTokens 限制得比較小。
  • 若您需要更長的回答 / 更多上下文，請依下方「自訂為自己的 Open AI API」一節，把端點換成自己的 Azure OpenAI 或 Open AI 官方 API。

目前預設為："MaxTokens": 200（最大不超過 4000）。

對話歷史保留最近 5 輪往復（5 次「我說 + AI 回」共 10 條作為上下文），早於此的對話會被丟棄以控制成本。例如：

我：你好
AI：你好，有什麼可以幫你
我：使命召喚是什麼遊戲？
AI：使命召喚是一款 FPS 遊戲。

以上是 2 輪。

其他要點：

  • 對話框內容支援 Markdown 渲染（連結、粗體、列表等）；點擊 AI 輸出的連結可跳到您站台其他頁面。
  • 自 1.9.16 起，提示氣泡會向上擴展而不是向下，方便顯示長回覆。
  • 外掛提供的免費通道目前伺服器在上海，無法直連 OpenAI 官方介面；作者使用的是個人 Azure OpenAI 通道。

自訂為自己的 Open AI API

您可以透過後台設定將其修改為自己使用的 Open AI API。

目前支援 Azure OpenAI 與 Open AI 官方介面，可使用 gpt-3.5-turbo、GPT-3.5-Turbo-0613 與 GPT-4。

Azure OpenAI 設定

1. API Key 與 Endpoint（端點）

在 Azure 後台 Keys and Endpoint 中可取得 Key 與 Endpoint。

如下圖所示，Key 可以二選一；Endpoint 是您這個執行個體的名稱，而 不是 下方的區域端點 URL。

2. Model Name（部署的模型名稱）

進入 https://oai.azure.com/portal/

建立模型後可在 Deployments 中看到 Deployment Name。

將此名稱填入 Model Name（部署的模型名稱）欄即可。

3. Version（版本）

請參閱 Azure OpenAI 模型版本文件。

目前所示最新版本為 2023-05-01，可填此固定值。
