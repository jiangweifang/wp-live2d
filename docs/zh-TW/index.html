自行架設 V1 模型伺服器（外部）

由 fghrsh 維護的舊版 Live2D API（V1）。自 2.1.3 起，本外掛已內建常用 V1 模型，一般不再需要第三方 API。

注意事項

  1. 請自行保護模型版權。V1 模型（journey-ad / Cubism 2~3 系列）已流傳多年，本外掛僅做相容性載入；V2 模型（Cubism 4+，*.model3.json）建議放在阿里雲 OSS / 騰訊雲 COS 等具備防盜連 + 跨來源控制的物件儲存上，或在外掛設定中開啟「快取至本機」讓外掛代為防盜。
  2. 創意工坊目前不開放上傳，原因同上 —— 在 2.0 版本升級到可防盜用之前，無法保障各位上傳者的版權。
  3. 請控制您的模型貼圖大小，建議不超過 2048，否則載入時間會非常長。
  4. moc3 的縮放比例可能比較小。請將模型縮放比例調整至適當大小（建議 2.0 左右）。程式碼中沒有寫死 2.0,因為它會以倍數複合（1.1 實際會變 2.2）。
  5. 請務必確認您的 WordPress REST API（rest_route）能正常存取。儲存設定出錯時，請先檢查站台是否將 REST API 加入白名單；Cloudflare 使用者尤其常見。自 2.1.3 起，本機 V1 模型服務、V2 防盜連都依賴 /wp-json/live2d/v1/...。
  6. 使用 CDN 加速的站長，請記得在更新外掛後刷新快取。
  7. live2d.js（以及 waifu-tips.min.js 等）不需要再次壓縮。若壓縮後出現錯誤，請先排除二次壓縮的問題。
  8. 內網使用者請以公開 IP 存取您的網站，並設定好網域（可使用花生殼免費網域）。在 localhost 上外掛無法儲存設定。
  9. 您可以在 nizima 購買正版 MOC3 模型。
  10. 您可以在官方 Live2D Showcase 看到本外掛的介紹。
  11. 更新紀錄會在 微博 或 bilibili 公布。
  12. 付費後若無回應，請及時聯絡作者 QQ：85838607，會盡快為您處理。
