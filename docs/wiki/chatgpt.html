ChatGPT 的使用

感谢您使用我的插件。

功能与默认配额

  • 本插件中提供了 ChatGPT 的功能，您可以通过点击机器人和 ChatGPT 进行交互。但是由于 ChatGPT 是付费的，所以我在默认代码中控制了 ChatGPT 的 Token 数量。

当前设置为："MaxTokens": 200

历史对话为 10 条往复的 Context（上下文），5 条包含自己说的：

例如：

  • 我：你好
  • AI：你好，有什么可以帮你
  • 我：使命召唤是一个什么游戏？
  • AI：使命召唤是一个 FPS 游戏。

以上对话是 2 条。

自定义为自己的 Open AI API

您可以通过后台设置将其修改为自己使用的 Open AI API。

目前支持 Azure OpenAI 和 Open AI 官方接口，可以使用 gpt-3.5-turbo、GPT-3.5-Turbo-0613 和 GPT-4。

Azure OpenAI 设置

1. Api Key 和 Endpoint（节点）

在 Azure 后台 Keys and Endpoint 中可以获取 Key 和 Endpoint。

如下图所示，Key 可以二选一；Endpoint 终结点是您这个实例的名称，而 不是 下面那个节点网址。

2. Model Name（部署的模型名称）

进入 https://oai.azure.com/portal/

创建模型后可以在 Deployments 中看到 Deployment Name。

将此名称放入 Model Name（部署的模型名称）中即可。

3. Version（版本）

请参阅 Azure OpenAI 模型版本文档。

当前所示的最新版本为 2023-05-01，可以填写此固定值。
