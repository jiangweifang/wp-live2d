5. 模型版权（站长 / 用户须知）​

本项目本身不分发任何带版权的成品 Live2D 模型，但允许用户加载第三方模型。请务必注意：

  1. V1 模型（*.moc，journey-ad / Cubism 2~3 系列）多数早期模型已经流传多年；插件提供的是兼容性加载，不代表插件作者拥有这些模型的版权。请尊重原作者署名，商业使用请联系原作者获取授权。
  2. V2 模型（*.model3.json，Cubism 4+）多数为商品模型，请到 nizima 等正规渠道购买使用许可。严禁 未经授权地分发、二次售卖他人创作的 MOC3 模型。
  3. 创意工坊目前不开放上传 —— 在 2.0 版本升级到可以防盗用的版本之前，无法保护各位上传者的版权。
  4. 建议：将自购模型放在带防盗链 + 跨域控制的对象存储里，或在插件设置中启用 托管到本站。
