ChatGPT 的使用

感谢您使用我的插件。

功能与默认配额

  • 本插件中提供了 ChatGPT 的功能，您可以通过点击机器人和 ChatGPT 进行交互。由于 ChatGPT 是付费 API，作者免费提供的默认通道有配额限制，不希望被滥用，所以默认代码里把 MaxTokens 限制得比较小。
  • 如果你需要更长的回答 / 更多上下文，请按下面「自定义为自己的 Open AI API」一节，把接口换成自己的 Azure OpenAI 或 Open AI 官方 API。

当前默认设置为："MaxTokens": 200（最大不超过 4000）。

历史对话保留最近 5 轮往复（5 次「我说 + AI 回」共 10 条作为上下文），早于此的对话会被丢弃以控制成本。例如：

我：你好
AI：你好，有什么可以帮你
我：使命召唤是一个什么游戏？
AI：使命召唤是一个 FPS 游戏。

以上是 2 轮。

其他要点：

  • 对话框里的内容支持 Markdown 渲染（链接、加粗、列表等）；点击 AI 输出的链接可以跳到你站点其他页面。
  • 从 1.9.16 起，提示气泡会向上扩展而不是向下，方便显示长回复。
  • 插件提供的免费通道目前服务器在上海，无法直连 OpenAI 官方接口；作者用的是个人 Azure OpenAI 通道。

自定义为自己的 Open AI API

您可以通过后台设置将其修改为自己使用的 Open AI API。

目前支持 Azure OpenAI 和 Open AI 官方接口，可以使用 gpt-3.5-turbo、GPT-3.5-Turbo-0613 和 GPT-4。

Azure OpenAI 设置

1. Api Key 和 Endpoint（节点）

在 Azure 后台 Keys and Endpoint 中可以获取 Key 和 Endpoint。

如下图所示，Key 可以二选一；Endpoint 终结点是您这个实例的名称，而 不是 下面那个节点网址。

2. Model Name（部署的模型名称）

进入 https://oai.azure.com/portal/

创建模型后可以在 Deployments 中看到 Deployment Name。

将此名称放入 Model Name（部署的模型名称）中即可。

3. Version（版本）

请参阅 Azure OpenAI 模型版本文档。

当前所示的最新版本为 2023-05-01，可以填写此固定值。
