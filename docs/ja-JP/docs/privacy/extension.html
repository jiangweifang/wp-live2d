ブラウザ拡張機能のプライバシーポリシー

対象製品：Chrome / Edge ブラウザ拡張機能 Live2D Web Canvas（BETA テスト版を含む） 発効日：2025-01-01 ｜ 最終更新：2026-05-05 発行主体：北京市天先生軟件技術中心

本ポリシーは Chrome / Edge ブラウザ拡張機能 Live2D Web Canvas（以下「本拡張機能」）が、ユーザー使用中にデータをどのように扱うかを説明します。本ポリシーは 本サイトの他のサービス、WordPress プラグイン、その他のクライアント製品 には適用されません。

1. 収集するデータ​

本拡張機能は Chrome ウェブストアの「データ使用」分類に従い、以下の 2 種類のみを取り扱い、ユーザーが自発的にログインした後にのみ発生します：

データ種別具体的な内容収集タイミング
認証情報本サイトが発行するアクセストークン（JWT、key と sign を含む）。有料機能のロック解除に使用。ユーザーが拡張機能で「ログイン」をクリックし、www.live2dweb.com のログインページで資格情報を提出後、拡張機能が受信。
個人識別情報ユーザーアカウントと一対一に対応する内部識別子（ログイン時のアカウントに紐付け）。本拡張機能は氏名、メールアドレス、電話、年齢、身分証番号、住所などその他個人を識別可能な情報を 収集しません。上記同様：ログイン後にアクセストークンと共に取得されるのみ。

収集 しない データ（明示的宣言）​

  • ユーザーが訪問するページの URL、タイトル、閲覧履歴を読み取り・保存・アップロードしません。
  • 訪問ページのテキスト、画像、フォーム、入力内容、Cookie、DOM ノードを読み取り・アップロードしません。
  • マウスクリック、スクロール、キー入力を収集しません（マスコットは現在のカーソル位置に基づくローカルでの向き合いアニメーションのみ行い、データはブラウザ外に出ません）。
  • 位置情報、IP アドレス、デバイスフィンガープリントを収集しません。
  • 健康、医療、財務、信用、決済カード番号などの機微情報を収集しません。
  • メール、インスタントメッセージ、その他あらゆる形式の個人通信を収集しません。

2. データの保存場所​

保存場所保存内容
chrome.storage.sync
（ローカル、Chrome アカウントに同期）		拡張機能の設定：マスコットの有効化、選択モデル、選択テクスチャ、ヒント発火頻度、ツールバーの切替など。
chrome.storage.local
（ローカルのみ）		ログイン後に取得したアクセストークン（JWT）。デバイス間で同期されません。
ブラウザ IndexedDB（ローカル）ユーザーがモデルライブラリから自発的にダウンロードした Live2D モデルファイル（.moc、PNG テクスチャ、モーション JSON など）。オフラインロード・高速表示に使用。
本サイトのバックエンドサーバー https://api.live2dweb.comユーザーアカウント自体の情報（本サイトログイン時に既に存在し、本拡張機能とは無関係）。および拡張機能が API を呼び出す際にアクセストークンと共に送られる通常のアクセスログ（時刻、リクエストパス、応答コード）。ページ内容は含みません。

3. データ利用（単一目的）​

本拡張機能の単一目的は：ウェブページの右下に Live2D 仮想キャラクターをオーバーレイすること。収集データは 以下のみ に使用されます：

  1. 課金済みユーザーを識別し、拡張機能内で「カスタムモデル URL」など有料機能を使えるようにする。
  2. 本サイト自有ドメイン https://download.live2dweb.com から、ユーザーが選択したモデルファイルをダウンロードする。
  3. ユーザーの表示設定を保存し、異なるタブ・デバイス間で一貫した体験を提供する。

4. データの共有と移転​

  • ユーザーデータを 販売しません。
  • 本拡張機能の単一目的に無関係な第三者にデータを 移転しません。
  • 信用評価、貸付、その他あらゆる与信目的に 使用しません。
  • 本サイト自有ドメイン（www.live2dweb.com、api.live2dweb.com、download.live2dweb.com）以外の第三者サーバーにリクエストを発行しません。

5. ユーザーによる自データの管理​

  • ログアウト：拡張機能のオプションページ（拡張機能アイコン → オプション → ログインタブ）で「ログアウト」をクリックすれば、ローカルのアクセストークンが消去されます。
  • モデルキャッシュのクリア：オプションページの「V1 モデルライブラリ」タブで個別にダウンロード済みモデルを削除し、IndexedDB の容量を解放できます。
  • すべての設定をクリア：chrome://extensions で本拡張機能をアンインストールすると、ブラウザは chrome.storage と IndexedDB に保存された本拡張機能のすべてのデータを一括削除します。
  • アカウント削除 / サーバー側個人情報の削除：下方の「お問い合わせ」メールへ削除依頼をお送りください。合理的な期間内に対応します。

6. データセキュリティ​

  • アクセストークンは HTTPS で送信され、HMAC 署名で改ざん検証されます。
  • トークンはユーザーローカルブラウザの chrome.storage.local にのみ保存され、URL やログには現れません。
  • 本拡張機能は リモートホストコード（Remote Hosted Code）を一切使用しません：すべての JavaScript は拡張機能パッケージに同梱されており、download.live2dweb.com から取得するのはモデルデータリソース（バイナリ / 画像 / JSON）のみで、コードとして実行されることはありません。

7. 児童プライバシー​

本拡張機能は 13 歳未満の児童を対象としておらず、児童の個人情報を意図的に収集しません。

8. ポリシーの変更​

本ポリシーに重大な変更があった場合、ページ上部の「最終更新」日付を更新し、拡張機能の更新説明を通じてユーザーに通知します。継続して使用することは、変更後のポリシーへの同意とみなされます。
