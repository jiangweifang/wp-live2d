注意事项

  1. 请自行保护模型版权，将会在 2.0 版本升级到可以防盗用的版本，在那之前，创意工坊不提供上传功能。
  2. 您可以通过上传到阿里云 OSS 或者腾讯云 COS，通过防盗链和跨域保护来保证自己的模型不被盗用。
  3. 您需要控制您的模型材质，不要过大，建议到 2048 即可，否则加载模型的时间会很长。
  4. moc3 的缩放比例可能会比较小，所以您需要将模型缩放比例调整至合适大小，建议 2.0。我没有进行控制是因为如果在代码中写入 2.0，其计算方式将会 ×2，例如 1.1 实际计算结果将会是 2.2。
  5. 请务必保证您的 WordPress REST API（rest_route）可以被正常访问。如果您在保存时出现异常，请检查站点是否将 WordPress REST API 设置了白名单；Cloudflare 用户很有可能出现这种情况。
  6. 使用 CDN 加速的朋友请注意，更新插件后记得刷新 CDN 加速缓存。
  7. live2d.js 无需 JS 压缩。如果压缩后出现错误，请自行查看是否已排除。
  8. 内网用户请使用公网 IP 访问您的网站，并配置好域名（可以是花生壳免费域名）。本地测试时插件无法保存。
  9. 您可以在 nizima 中购买到正版 MOC3 模型。
  10. 您可以在官方 Live2D Showcase 中看到本插件的介绍。
  11. 更新日志将会在 微博 或 bilibili 更新。
  12. 付费后如果没有反应，请及时联系作者 QQ：85838607，会尽快为您解决。
