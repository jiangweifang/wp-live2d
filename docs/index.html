注意事项

  1. 请自行保护模型版权。V1 模型（journey-ad / Cubism 2~3 系列）已经流传多年，本插件提供的是兼容性加载；V2 模型（Cubism 4+，*.model3.json）建议放在阿里云 OSS / 腾讯云 COS 等带防盗链 + 跨域控制的对象存储里，或在插件设置中开启「缓存到本地」让插件代为防盗。
  2. 创意工坊目前不开放上传，原因同上 —— 在 2.0 版本升级到可以防盗用的版本之前，无法保护各位上传者的版权。
  3. 请控制您的模型材质大小，建议不超过 2048，否则加载时间会很长。
  4. moc3 的缩放比例可能会比较小。请将模型缩放比例调整至合适大小（建议 2.0 左右）。代码中没有写死 2.0，是因为它会按倍数复合（1.1 实际会变成 2.2）。
  5. 请务必保证您的 WordPress REST API（rest_route）可以被正常访问。如果保存设置时出错，请检查站点是否将 REST API 加了白名单；Cloudflare 用户尤其常见。从 2.1.3 起，本地 V1 模型服务、V2 防盗链都依赖 /wp-json/live2d/v1/... 路径。
  6. 使用 CDN 加速的朋友请注意，更新插件后记得刷新缓存。
  7. live2d.js（以及 waifu-tips.min.js 等）无需再次压缩。如压缩后出现错误，请先排除二次压缩。
  8. 内网用户请使用公网 IP 访问您的网站，并配置好域名（可以是花生壳免费域名）。本地测试时插件无法保存。
  9. 您可以在 nizima 中购买到正版 MOC3 模型。
  10. 您可以在官方 Live2D Showcase 中看到本插件的介绍。
  11. 更新日志将会在 微博 或 bilibili 更新。
  12. 付费后如果没有反应，请及时联系作者 QQ：85838607，会尽快为您解决。
