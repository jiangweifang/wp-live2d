{%- comment -%} Builds an absolute path inside the docs site. Inputs (Liquid include vars): * path — a leading-slash path like "/wiki/x.html" or "#guides". * lang — current page language code ('zh-CN', 'en', ...). For lang == 'zh-CN' the path is returned as-is. For other langs, paths starting with '/' are prefixed with '/'. Anchors and protocol-absolute URLs are returned unchanged. {%- endcomment -%} {%- assign __first = include.path | slice: 0, 1 -%} {%- if include.lang == 'zh-CN' or __first != '/' -%}{{ include.path }} {%- else -%}/{{ include.lang }}{{ include.path }} {%- endif -%}
