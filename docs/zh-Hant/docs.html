Live2D Web Canvas · 文件中心

本站收錄 Live2D Web Canvas 的使用文件。Live2D Web Canvas 同時維護一個 WordPress 外掛和一個 Chrome / Edge 瀏覽器擴充功能。

請使用左側導覽列依下列分類查閱：

  • 基本介紹：先搞清楚 V1 / V2 模型的差異。
  • 基本設定：modelAPI / modelDir 的填寫方式、V1 / V2 模型的兩種使用方式、V2 防盜連。
  • 進階設定：自訂 ChatGPT 端點、V2 模型 HitArea 觸控區域。
  • Chrome / Edge 擴充功能：Web Store 安裝位址、移除方式與登入設定。
  • 隱私 & 著作權：擴充功能隱私政策與專案著作權 / 開源授權條款。

一般注意事項​

  1. 請控制您模型的材質大小，建議不超過 2048×2048，否則載入時間會很長。
  2. moc3 的縮放比例可能會比較小，建議在後台把模型縮放調到 2.0 左右。程式碼不寫死 2.0 是因為它會按倍數複合（設 1.1 實際會變成 2.2）。
  3. 請確保您的 WordPress REST API（rest_route）能被正常存取。如果儲存設定時出錯，請先檢查 REST API 是否被安全外掛 / Cloudflare 攔截。從 2.1.3 起，本機 V1 模型服務、V2 防盜連都依賴 /wp-json/live2d/v1/... 路徑。
  4. 使用 CDN 加速的朋友請注意，更新外掛後記得清除快取。
  5. live2d.js / waifu-tips.min.js 等 無需再次壓縮，二次壓縮可能會引發錯誤。
  6. 內網 / 本機測試時外掛無法儲存設定，請使用公開 IP + 網域（可以是免費 DDNS 網域）存取站點。
  7. 您可以在 nizima 購買正版 MOC3 模型；本外掛也已被收錄在 Live2D 官方 Showcase。
  8. 更新日誌會在 微博 與 bilibili 發佈。
  9. 付費後若沒有反應，請加入 QQ 群：49463654。（僅限中文）
