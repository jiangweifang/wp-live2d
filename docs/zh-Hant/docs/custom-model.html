使用 V2 自訂模型（Cubism 4+）

本頁只談 V2（Cubism 4+ / 5）模型。若您想用 22/33/Pio/Tia 之類的舊模型，那是 V1，請走外掛後台「創意工坊 API 類型 = 本機 / 遠端創意工坊」，不需要付費，也不需要本頁任何步驟。請先看 V1 / V2 是什麼。

1. 先付費

V2 模型走的是 Live2D Cubism 4+ Web SDK 載入器。Live2D 株式會社要求第三方擴充型應用使用 Cubism 4+ 時必須簽訂商業合約，本外掛作者已簽訂長期合約並依其要求散布正版 Cubism Core，所以這條路徑會向站長收取一次性授權費。

  1. 於 www.live2dweb.com 完成註冊與付費。
  2. 於 WordPress 後台保持登入狀態（這一步是為了取得與您帳號綁定的 token，讓外掛能向 live2dweb.com 驗證授權）。

未付費 / 未登入時，後台「創意工坊 API 類型」中的 自訂 選項會被停用或提示需要登入。

2. 準備好您的 V2 模型檔案

V2 模型由 Cubism 4+ 匯出，清單檔是 *.model3.json，本體為 *.moc3，再加上貼圖、動作、表情等。

2.1 清單檔需求

  • 清單檔名一般為 {模型名}.model3.json。
  • 路徑中 不能含中文字。
  • 若您的站台是 https，那麼清單以及它引用到的所有檔案（.moc3、貼圖、motion3.json、physics3.json、cdi3.json 等）都必須是 https，否則瀏覽器會以 Mixed Content / 跨來源錯誤拒絕。
  • 跨來源：把模型放在 OSS / COS / 自家 CDN 時，請正確設定 CORS（通常加入您的站台網域為允許 Origin 即可）。

2.2 模型放在哪裡？

依推薦度高到低：

  1. 物件儲存 + 防盜連（首選，自帶版權保護）：阿里雲 OSS、騰訊雲 COS、又拍、七牛等。
    • 啟用 Referer 白名單，只允許您的站台存取。
    • 套上自家 CDN 時，請記得讓 CDN 端繼承相同的防盜連規則。
  2. 外掛目錄下的 model 資料夾（最簡單，但流量得自己控制）： wp-content/plugins/live-2d/model/{模型名}/{模型名}.model3.json 若沒有 model 資料夾，請自行建立。注意 WordPress 的 URL Rewrite 在某些佈景主題 / 安全外掛下可能會誤攔截，存取不到時請先確認該 URL 能在瀏覽器中直接開啟。
  3. 啟用「快取至本機」讓外掛接管：填好 modelAPI 後，把防盜連選項切到「快取至本機」，外掛會把對應檔案全部下載到您站台的 wp-content/plugins/live-2d/model/，並對前端只露出暫時性的簽名 URL，訪客 F12 也看不到原始位址。詳見防盜連頁。
  4. 放在 GitHub Pages / jsDelivr 這類公開靜態託管：能跑，但不建議在中國大陸地區使用，jsDelivr 最終仍會回源至 raw.githubusercontent.com，速度不穩，也無法防盜。

3. 在後台填入 modelAPI

WordPress 後台 → Live2D 設定 → 創意工坊 API 類型選「自訂」。下方有 modelAPI 輸入框，有兩種寫法，由結尾字元決定走哪一種：

寫法結尾作用下方 modelDir 列表
① 直連單模型以 .json 結尾直接載入該 model3.json，不再做任何路徑串接。忽略
② 目錄根 + 多模型以 / 結尾視為根目錄，搭配下方 modelDir 列表，依 {modelAPI}{名稱}/{名稱}.model3.json 自動串成完整 URL。必填，每行一個子目錄名（須與 {名稱}.model3.json 完全一致）。

範例

① 直連：

https://cdn.example.com/live2d/haru/haru.model3.json

② 目錄根 + 多模型：

modelAPI:  https://cdn.example.com/live2d/

modelDir:


    

  • haru
    • 

  • mao
    •
  • shizuku

實際載入時外掛會展開為：

https://cdn.example.com/live2d/haru/haru.model3.json
https://cdn.example.com/live2d/mao/mao.model3.json
https://cdn.example.com/live2d/shizuku/shizuku.model3.json

前台「眨眼按鈕」會在這幾個模型之間切換。

注意：使用自訂模型後，前台 Live2D 的「上一個 / 下一個」切換按鈕在許多模型組合下沒有意義，建議在後台關閉這兩顆按鈕。

4. 模型貼圖（材質）大小

請控制每張貼圖不超過 2048×2048。Cubism 4+ 預設貼圖就比較大，疊上幾張過大的 PNG 很容易把首屏載入時間拖到 5~10 秒以上。若模型本身匯出時貼圖解析度超標，可在 Cubism Editor 中「嵌入紋理圖集 → 重新分配尺寸」。

5. 進階：超採樣（僅 V2）

外掛 2.1.3 起新增 超採樣功能，可讓貼圖在高 DPI 螢幕上更清晰。僅對 V2（model3.json）模型生效，V1 路徑不會回應這個開關。

6. 安全 / 版權要點

  1. 請自行保證自家模型的版權。最佳做法：上傳到具防盜連的物件儲存 + 啟用 CORS 限定來源。
  2. 創意工坊目前不支援自行上傳 —— 因為我們無法替每位上傳者保護版權。
  3. 不要把 OSS 的 SecretKey 寫進 WordPress 設定項。若一定要走 Private 讀 + 簽名連結，請走 wp-config.php 環境變數，參見防盜連頁底部說明。

若有疑問，歡迎在 GitHub Issues 回報。
