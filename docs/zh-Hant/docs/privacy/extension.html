瀏覽器擴充功能隱私權政策

Live2D Web Canvas 瀏覽器擴充功能隱私權政策

適用產品：Chrome / Edge 瀏覽器擴充功能 Live2D Web Canvas（含 BETA 測試版）
生效日期：2025-01-01  |  最近更新：2026-05-05
發行主體：北京市天先生軟體技術中心

本政策僅說明 Chrome / Edge 瀏覽器擴充功能 Live2D Web Canvas（以下稱「本擴充功能」） 在使用者使用過程中如何處理資料。本政策不適用於本網站的其他服務、WordPress 外掛或其他用戶端產品。

1. 我們收集哪些資料

本擴充功能依 Chrome Web Store「資料使用」分類，僅處理以下兩類資料，且僅在使用者主動登入後才會發生：

資料類型具體內容收集觸發時機
身分驗證資訊本網站簽發的存取令牌（JWT，含 key 與 sign），用於解鎖付費功能。使用者在擴充功能中點擊「登入」，並在 www.live2dweb.com 登入頁送出帳號密碼後，由擴充功能接收。
個人身分資訊與使用者帳號一一對應的內部識別碼（與登入時的帳號關聯）。本擴充功能不收集姓名、電子郵件、電話、年齡、身分證件號、郵遞地址等其他可識別個人的資訊。同上：僅在登入後隨存取令牌一起取得。

我們不收集的資料（明確聲明）

  • 不讀取、不儲存、不上傳您造訪的網頁 URL、網頁標題或瀏覽歷史。
  • 不讀取、不上傳所在網頁的任何文字、圖片、表單、輸入內容、Cookie 或 DOM 節點。
  • 不收集您的滑鼠點擊、捲動、按鍵操作（看板娘只在本機根據目前滑鼠位置做朝向動畫，資料不離開您的瀏覽器）。
  • 不收集地理位置、IP 位址、裝置指紋。
  • 不收集健康、醫療、財務、信用、付款卡號等任何敏感資訊。
  • 不收集電子郵件、即時訊息或任何形式的個人通訊。

2. 資料儲存於何處

儲存位置存放內容
chrome.storage.sync
本機，並隨您的 Chrome 帳號同步		擴充功能偏好設定：是否啟用看板娘、所選模型、所選貼圖、提示氣泡頻率、工具列開關等。
chrome.storage.local
僅本機		登入後取得的存取令牌（JWT）。該令牌不會跨裝置同步。
瀏覽器 IndexedDB（本機）使用者主動從模型庫下載的 Live2D 模型檔案（.moc、PNG 貼圖、動作 JSON 等），用於離線載入與加速顯示。
本網站後端伺服器
https://api.live2dweb.com		使用者帳號本身的資訊（在使用者登入本網站時即已存在，與擴充功能無關）；以及擴充功能呼叫 API 時附帶的存取令牌的常規存取記錄（含存取時間、請求路徑、回應碼），不含網頁內容。

3. 資料用途（單一用途）

本擴充功能的單一用途是：在網頁右下角疊加一個 Live2D 虛擬形象。上述被收集的資料僅用於：

  1. 識別已付費使用者，使其能在擴充功能中使用「自訂模型 URL」等付費功能；
  2. 從本網站自有網域 https://download.live2dweb.com 下載使用者選擇的模型檔案；
  3. 儲存使用者的顯示偏好，使其在不同分頁與不同裝置間維持一致體驗。

4. 資料分享與轉讓

  • 我們不會出售使用者資料。
  • 我們不會將使用者資料轉交給與本擴充功能單一用途無關的第三方。
  • 我們不會將使用者資料用於評估信用、放貸或任何與信貸相關的用途。
  • 除存取本網站自有網域（www.live2dweb.com、api.live2dweb.com、download.live2dweb.com）外，本擴充功能不向任何第三方伺服器發起請求。

5. 使用者對自身資料的控制

  • 登出：在擴充功能選項頁（點擊擴充功能圖示 → 選項 → 登入標籤）點擊「退出登入」，即可清除本機保存的存取令牌。
  • 清空模型快取：在選項頁「V1 模型庫」標籤可逐項刪除已下載的模型，從 IndexedDB 釋放儲存空間。
  • 清空所有偏好：在 chrome://extensions 中解除安裝本擴充功能，瀏覽器會一併清除 chrome.storage 與 IndexedDB 中本擴充功能的所有資料。
  • 登出帳號 / 刪除伺服器端個人資料：請透過下方「聯絡我們」電子郵件寄送刪除請求，我們將在合理時間內處理。

6. 資料安全

  • 存取令牌透過 HTTPS 傳輸，並以 HMAC 簽名校驗防止竄改。
  • 令牌僅保存在使用者本機瀏覽器的 chrome.storage.local，不會出現在 URL 或記錄中。
  • 本擴充功能不使用任何遠端託管程式碼（Remote Hosted Code）：所有 JavaScript 均隨擴充功能安裝包散布，從 download.live2dweb.com 取得的內容僅為模型資料資源（二進位 / 圖片 / JSON），不會被當作程式碼執行。

7. 兒童隱私

本擴充功能不針對 13 歲以下兒童設計，亦不會有意識地收集兒童的個人資訊。

8. 政策變更

若本政策有重大變更，我們會更新頁面頂部的「最近更新」日期，並透過擴充功能更新說明告知使用者。繼續使用本擴充功能即視為接受變更後的政策。
