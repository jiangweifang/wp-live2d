5. Model copyright (notice for site owners and users)​

The project itself does not distribute any copyrighted ready-made Live2D models, but it does allow users to load third-party models. Please note:

  1. V1 models (*.moc, journey-ad / Cubism 2~3 series): many early models have been circulating for years; the plugin only provides compatibility loading and does not represent the author owning those models. Please respect the original authors' attribution, and contact them for commercial-use licensing.
  2. V2 models (*.model3.json, Cubism 4+): most are commercial models; please buy the use license through legitimate channels such as nizima. It is strictly forbidden to distribute or resell other people's MOC3 models without authorization.
  3. The workshop currently does not allow uploads — until version 2.0 reaches a hotlink-protected state, we cannot protect uploaders' copyright.
  4. Recommendation: keep models you've purchased on object storage with anti-hotlink + cross-origin restrictions, or enable Host on this site in the plugin settings.
