Thanks for using the plugin.

Feature and default quota

  • The plugin includes a ChatGPT feature: clicking the character lets you chat with ChatGPT. Because the ChatGPT API is paid, the free channel the author provides has a quota and we do not want it abused, so by default MaxTokens is kept fairly small.
  • If you want longer answers / more context, follow the section "Plug in your own Open AI API" below to swap the endpoint to your own Azure OpenAI or official Open AI API.

The current default is "MaxTokens": 200 (cap is 4000).

The conversation history keeps the most recent 5 turns ("you say + AI replies" — 10 messages total) as context; anything older is dropped to control cost. For example:

Me: hi
AI: hello, how can I help?
Me: what kind of game is Call of Duty?
AI: Call of Duty is an FPS game.

That's 2 turns.

Other notes:

  • Messages in the bubble support Markdown rendering (links, bold, lists, …); clicking links produced by the AI navigates to other pages on your site.
  • Since 1.9.16, the tip bubble expands upward instead of downward, which makes long replies easier to read.
  • The free channel's server is currently in Shanghai and cannot reach the official OpenAI API directly; the author uses a personal Azure OpenAI channel.

Plug in your own Open AI API

You can switch to your own Open AI API on the backend settings page.

Both Azure OpenAI and the official Open AI API are supported, including gpt-3.5-turbo, GPT-3.5-Turbo-0613 and GPT-4.

Azure OpenAI setup

1. API Key and Endpoint

The Key and Endpoint can be found under Keys and Endpoint in the Azure portal.

As shown below, either Key works; the Endpoint is the name of this instance, not the regional endpoint URL listed beneath it.

2. Model Name (deployment name)

Go to https://oai.azure.com/portal/.

After creating a model, the Deployment Name appears under Deployments.

Put that name into the Model Name (deployment name) field.

3. Version

See the Azure OpenAI model versions doc.

At time of writing the latest version is 2023-05-01; you can use that fixed value.
