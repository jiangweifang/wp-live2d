Live2D Web Canvas ブラウザ拡張 — プライバシーポリシー

対象製品:Chrome / Edge ブラウザ拡張 Live2D Web Canvas(BETA 版を含む)
発効日:2025-01-01  |  最終更新:2026-05-05
発行主体:北京市天先生软件技术中心

本ポリシーは、Chrome / Edge ブラウザ拡張 Live2D Web Canvas(以下「本拡張」) がユーザー利用時にどのようにデータを取り扱うかを説明するものです。本ポリシーは適用されません: 本サイトの他のサービス、WordPress プラグイン、その他のクライアント製品。

1. 取得するデータ

本拡張は Chrome Web Store の「データ使用」分類に従い、以下の 2 種類のデータのみを取り扱います。いずれもユーザーが明示的にログインした後にのみ発生します:

種別内容取得タイミング
認証情報本サイトが発行するアクセストークン(key と sign を含む JWT)。有償機能を解錠するために使用します。ユーザーが拡張内で「ログイン」をクリックし、www.live2dweb.com のログインページで資格情報を送信した後、拡張側で受信。
個人を識別できる情報ユーザーアカウントに 1 対 1 で対応する内部識別子(ログイン時のアカウントに紐付き)。本拡張は氏名、メールアドレス、電話番号、年齢、政府発行 ID、郵便住所等の他の個人識別情報を 収集しません。同上:ログイン後にアクセストークンと共に取得。

明示的に 収集しない データ

  • 閲覧したページの URL、タイトル、閲覧履歴を読み取り・保存・送信しません。
  • ホストページのテキスト、画像、フォーム入力、Cookie、DOM ノードを読み取り・送信しません。
  • マウスクリック、スクロール位置、キー入力を収集しません(キャラはローカルでマウス位置に応じて動くだけで、データはブラウザから出ません)。
  • 位置情報、IP アドレス、デバイス指紋を収集しません。
  • 健康・医療・金融・与信・カード番号等の機微情報を一切収集しません。
  • メール、チャット等の個人通信を収集しません。

2. データの保存場所

場所内容
chrome.storage.sync(ローカル + Chrome アカウント間で同期)拡張の設定:有効/無効、選択中のモデル、選択中のテクスチャ、提示バブル頻度、ツールバートグル等。
chrome.storage.local(ローカルのみ)ログイン後に取得したアクセストークン(JWT)。デバイス間で同期されません。
ブラウザ IndexedDB(ローカル)ユーザーがモデルライブラリから明示的にダウンロードした Live2D モデルファイル(.moc、PNG テクスチャ、モーション JSON 等)。オフライン利用と高速化のために保持します。
本サイトのバックエンド https://api.live2dweb.comユーザーアカウント本体(ユーザーがサイト登録時に作成済みで、本拡張とは無関係)。加えて、トークン付きリクエストの標準アクセスログ(タイムスタンプ、リクエストパス、レスポンスコード)。サーバーログにページ内容は含まれません。

3. データの用途(単一目的)

本拡張の単一目的は、ウェブページ右下に Live2D キャラクターをオーバーレイ表示することです。上記データは 以下の目的のみ に使用されます:

  1. 有償ユーザーを識別し、「カスタムモデル URL」等の有償機能を解錠する;
  2. ユーザーが選択したモデルファイルを自社ドメイン https://download.live2dweb.com からダウンロードする;
  3. ユーザー設定をタブ間およびデバイス間で保持する。

4. 共有および移転

  • ユーザーデータを販売しません。
  • 本拡張の単一目的に関係のない第三者にデータを移転しません。
  • 信用評価や貸付目的でデータを利用・移転しません。
  • 自社ドメイン(www.live2dweb.com、api.live2dweb.com、download.live2dweb.com)以外、本拡張は第三者サーバーへ通信しません。

5. 自分のデータへのコントロール

  • サインアウト:拡張のオプションページ(拡張アイコン → オプション → ログインタブ)で「サインアウト」をクリックすると、ローカル保存のアクセストークンが削除されます。
  • モデルキャッシュの削除:オプションページ「V1 ライブラリ」タブで、ダウンロード済みモデルを個別に削除できます。IndexedDB の容量を解放します。
  • すべて削除: chrome://extensions から本拡張をアンインストールすると、ブラウザは chrome.storage と IndexedDB の本拡張データを一括削除します。
  • アカウント削除 / サーバー側データ削除:下記「お問い合わせ」のメールアドレスへ削除依頼をお送りください。合理的な期間内に対応いたします。

6. データセキュリティ

  • アクセストークンは HTTPS で送信し、HMAC 署名で改ざんを検知します。
  • トークンはユーザーローカルブラウザの chrome.storage.local にのみ保存され、URL やログには現れません。
  • 本拡張は Remote Hosted Code を一切使用しません。すべての JavaScript は拡張パッケージに同梱されており、download.live2dweb.com から取得するのはモデルデータ(バイナリ / 画像 / JSON)のみで、コードとして実行されることはありません。

7. 子どものプライバシー

本拡張は 13 歳未満の児童を対象としておらず、児童の個人情報を意図的に収集することはありません。

8. ポリシーの変更

重要な変更があった場合、ページ上部の「最終更新」日付を更新し、拡張のアップデートノートでお知らせします。本拡張の継続利用は、変更後ポリシーへの同意とみなします。
