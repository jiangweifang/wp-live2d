本プラグインをご利用いただきありがとうございます。

機能と既定の割当

  • 本プラグインは ChatGPT 機能を提供しており、キャラをクリックすることで ChatGPT と対話できます。ChatGPT は有償 API のため、作者が無料提供している既定チャネルには上限があり、悪用されたくないので既定の MaxTokens はかなり小さく設定してあります。
  • もっと長い回答 / もっと長いコンテキストが必要なら、下の「自分の Open AI API に差し替える」を参照して、エンドポイントを自身の Azure OpenAI または公式 Open AI API に切り替えてください。

現在の既定値は "MaxTokens": 200(上限は 4000)です。

履歴は直近 5 ターン(「私 + AI」5 往復、計 10 メッセージ)を保持し、それより古いものはコスト抑制のため破棄されます。例:

私:こんにちは
AI:こんにちは、何をお手伝いしましょう?
私:Call of Duty とはどんなゲームですか?
AI:Call of Duty は FPS ゲームです。

これで 2 ターンです。

その他のポイント:

  • バブル内のテキストは Markdown 描画(リンク・太字・リスト等)に対応。AI が出力したリンクをクリックすると、自サイト内の他ページに遷移できます。
  • 1.9.16 以降、提示バブルは下方向ではなく 上方向 に伸びるため、長い回答が読みやすくなりました。
  • 無料チャネルのサーバは現在上海にあり、OpenAI の公式 API へ直接アクセスできません。作者は個人の Azure OpenAI チャネルを利用しています。

自分の Open AI API に差し替える

バックエンド設定ページ から、自分の Open AI API に切り替えることができます。

Azure OpenAI と公式 Open AI API の両方をサポートしており、gpt-3.5-turbo、GPT-3.5-Turbo-0613、GPT-4 などが使えます。

Azure OpenAI の設定

1. API Key とエンドポイント

Azure ポータルの Keys and Endpoint から Key と Endpoint を取得できます。

下図の通り、Key はどちらでも構いません。Endpoint はこのインスタンスの名前であり、その下にあるリージョナル URL ではありません。

2. Model Name(デプロイ名)

https://oai.azure.com/portal/ へアクセス。

モデル作成後、Deployments に Deployment Name が表示されます。

その名前を Model Name(デプロイ名)欄に入れてください。

3. Version

Azure OpenAI モデルバージョン ドキュメント を参照してください。

現時点での最新バージョンは 2023-05-01 で、この固定値を入力すれば構いません。
